Калькулятор (C++)



Опис проєкту

Цей проєкт створено для командної роботи з використанням Git.  

Програма реалізує простий калькулятор на мові C++, який виконує базові арифметичні операції.



Функціонал

\- Додавання

\- Віднімання

\- Множення

\- Ділення



Мета

Навчитись працювати з:

\- гілками (branches)

\- pull request

\- merge

\- вирішення конфліктів



Командна робота

Кожен учасник реалізує окрему функцію калькулятора у своїй гілці.
